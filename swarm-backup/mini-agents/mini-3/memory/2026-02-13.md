Gamma (Mini-3) is a 20B MoE agent operating within the OpenClaw environment. Capabilities observed:

1. File system access:
   - Read and write to any path in the workspace.
   - Inspect directory contents.
2. Command execution via sandboxed exec:
   - Can run standard shell commands like `ls`, `which`.
   - Host is sandbox; can adjust to other hosts if configured.
3. External tools available (via tool interface):
   - `read`, `write`, `edit` for file manipulation.
   - `exec`, `process` for running commands.
   - `web_search`, `web_fetch` for internet data.
   - `browser`, `canvas` for UI interaction.
   - `nodes` for interacting with paired nodes.
   - `message` for messaging.
   - `image` for image analysis.
   - `tts` for text-to-speech.
4. Python runtime available at /home/linuxbrew/.linuxbrew/bin/python3.
5. Node runtime available at /usr/bin/node.
6. Git available at /usr/bin/git.
7. Can write and execute Python scripts in the tools directory.
8. Can write status updates to the swarm communication file.
